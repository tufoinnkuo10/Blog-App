#  My Blog App

A fully functional blog app website that will show post of different users and allow the users to iteract with it through likes and comments.
## Built With

* Ruby on Rails
* Rspec
* Psql

## Getting Started

### Prerequisites

* Ruby (ruby --v)
* SQLite3 (sqlite3 --v)
* Postgrestsql (psql --version)
* Node.js (node --v)
* Yarn (yarn --v)

To get a local copy up and running follow these simple example steps.

Run in your terminal `git clone git@github.com:Mhdez221993/blog_app.git`

### Install RSpec

To run the tests locally :

* To instal RSpec for testing please run the following command on your terminal:

 ` gem install rspec`
